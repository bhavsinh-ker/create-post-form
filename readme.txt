== Description ==
This plugin will allow admin to add create post form in front-end. Once the plugin will be installed, you will be able to see create post form Gutenberg block.

You can add that Gutenberg block in the page or post and it will add [create_post_form] shortcode in the front-end.

You can also directly use [create_post_form] shortcode for display form on the front-end.

== Installation ==
- Go to Plugins -> Add New
- Find a plugin from the list or search for Create Post Form
- View more details about it before you decide to install the plugin
- Click on the \"Install\" button
- Wait for a few seconds until the orange icons stop spinning
- Click on the \"Active\" button
